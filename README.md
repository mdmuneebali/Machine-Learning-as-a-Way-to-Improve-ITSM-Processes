In this project, I analyzed historical ITSM incident data to identify patterns related to ticket categories, priorities, and resolution behavior. The dataset was first cleaned and preprocessed to make it suitable for analysis. Machine learning models were then applied to predict and classify service incidents, especially focusing on identifying critical issues early. These predictions can help IT teams respond faster, optimize resource allocation, and improve overall service efficiency.

The project demonstrates how machine learning can be practically used in ITSM environments to move from reactive issue handling to a more proactive and data-driven approach.
